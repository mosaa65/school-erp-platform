# System 02 - النواة الأكاديمية

## الهدف
بناء الهيكل الدراسي الذي تعتمد عليه بقية الأنظمة.

## المتطلبات المسبقة

1. اكتمال إعدادات System 01.
2. وجود صلاحيات `academic-years.*` و`academic-terms.*` و`sections.*`.

## التسلسل الموصى به

1. `Academic Years`:
   1. أنشئ السنة الدراسية وحدد الحالية.
2. `Academic Terms`:
   1. أضف الفصول المرتبطة بالسنة.
3. `Academic Months`:
   1. وزع الأشهر تحت الفصل/السنة.
4. `Grade Levels`:
   1. أضف المراحل/الصفوف بترتيبها.
5. `Subjects`:
   1. أضف المواد ومعلومات التصنيف.
6. `Sections`:
   1. أنشئ الشعب لكل صف.
7. `Grade Level Subjects`:
   1. اربط المواد بالصفوف.
8. `Term Subject Offerings`:
   1. اربط المادة بالفصل الأكاديمي.
9. `Timetable Entries`:
   1. اضبط الجدول الدراسي الفعلي.

## التحقق

1. لا تنشئ قيد طالب قبل وجود سنة + شعبة.
2. لا تنشئ تقييم/واجب قبل وجود offering صالح.

## سيناريو تشغيلي قياسي

1. إنشاء سنة أكاديمية جديدة وتفعيلها.
2. إنشاء فصلين دراسيين ضمن السنة.
3. إنشاء صف + شعبة + مادة.
4. ربط المادة بالصف ثم إنشاء offering للفصل.

## أخطاء شائعة وحلول

1. فشل إنشاء `term subject offering`:
   1. تأكد من وجود `grade-level-subject` لنفس السنة.
2. تعارض في الأكواد:
   1. استخدم code فريد لكل كيان (سنة/مادة/شعبة).
